Greed Command List

Prefix commands:
> selfprefix - sets your self prefix
> prefix or @greed - view your current prefix

Information commands:
> botinfo - sends information about greed
> ping - sends greed's ping
> help - info on how to invite greed/support server

Welcome system commands:
> welcomesystem - turns on/off welcome system
> welcomechannelset - sets channel to send welcome message
> welcomemessageset - sets message to send when user joins the server

Goodbye system commands:
> goodbyesystem - turns on/off goodbye system
> goodbyechannelset - sets channel to send goodbye message
> goodbyemessageset - sets message to send when user leaves the server

Moderation commands:
> ban - bans user from server
> kick - kicks user from server
> softban - bans user then unbans user from server
> unban - unbans user from server
> mute - mutes user in server
> unmute - unmutes user in server
> purge - deletes a certian amount of messages
